El centro cultural de San Miguel está en el barrio Centro y tiene 12 personas.
El centro cultural de Arbela está en el barrio San José y tiene 5 participantes.
El centro cultural de Altamira está en el barrio Mirador y cuenta con 8 estudiantes.
El centro cultural de Pancitará está en el barrio El libertador y agrupa a 15 personas.

¿Qué 2 tareas del backlog harías primero?
registrar a todos los estudiantes de cada corregimiento y realizar el codigo con las aplicaciones de apoyo.

¿Por qué son las más importantes?
 son la base para saber en donde y cuantas personas son beneficiarias de este proyecto  y generar la verificacion de que se esta desarrollando en el sitio y con todos los inscritos.

 ¿Qué significa para ti que una tarea esté “terminada”?
terminada se supone que ya no hay nada mas por hacer que tanto el backend como el frontend realizaron sus operaciones y fueron exitosas con ayuda de las aplicaciones como telegram u otros.

el codigo que esta escrito funciona de una buena forma y esta en server.js

// 1. Definición de la estructura de datos con la información suministrada
const centrosCulturales = [
  { nombre: "San Miguel", barrio: "Centro", cantidad: 12, tipoPersona: "personas" },
  { nombre: "Arbela", barrio: "San José", cantidad: 5, tipoPersona: "participantes" },
  { nombre: "Altamira", barrio: "Mirador", cantidad: 8, tipoPersona: "estudiantes" },
  { nombre: "Pancitará", barrio: "El libertador", cantidad: 15, tipoPersona: "personas" }
];

// 2. Iteración simple utilizando un bucle forEach
centrosCulturales.forEach(centro => {
  // Se determina el verbo adecuado según el tipo de dato para mantener la redacción original
  let conector = "tiene";
  if (centro.nombre === "Altamira") conector = "cuenta con";
  if (centro.nombre === "Pancitará") conector = "agrupa a";

  // Impresión de cada línea en la consola
  console.log(`El centro cultural de ${centro.nombre} está en el barrio ${centro.barrio} y ${conector} ${centro.cantidad} ${centro.tipoPersona}.`);
});
